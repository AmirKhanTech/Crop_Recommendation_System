# 🌱 Crop Recommendation System

An intelligent web application that recommends the **best crop to plant** based on **soil and weather conditions**.

This system helps **farmers, researchers, and agricultural enthusiasts** make data-driven decisions to improve productivity and maximize yield.

---

## 🚀 Features

* 🌾 Smart crop recommendation using:

  * Nitrogen (N), Phosphorus (P), Potassium (K)
  * Soil pH level
  * Temperature
  * Humidity
  * Rainfall

* 🧠 Machine Learning powered predictions

* 📊 Multiple models comparison and evaluation

* 🌐 Interactive UI built with **Streamlit**

* 🐳 Docker support for easy deployment

---

## 🗂 Dataset

This project uses the **Crop Recommendation Dataset** from Kaggle:

🔗 https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset

---

## 🧠 Machine Learning Models

All models are trained and evaluated in:

📁 `models.ipynb`

Models used:

* Random Forest
* Decision Tree
* K-Nearest Neighbors (KNN)
* Additional classification models

## ⚙️ Installation & Setup

### 🔹 Run Locally

```bash
# 1. Clone repository
git clone https://github.com/MPrevelato/Crop_Recommendation_System.git
cd Crop_Recommendation_System

# 2. Create virtual environment (recommended)
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run application
streamlit run app.py
```
